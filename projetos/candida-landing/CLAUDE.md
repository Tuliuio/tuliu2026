# Candida Landing Page

Projeto de landing page para Candida. Landing page responsiva com design moderno e conversão focada.

**Stack:** React + TypeScript + Vite + Tailwind CSS

---

## Estrutura do projeto

```
candida-landing/
├── projeto/           — código-fonte (React app)
├── briefing/          — briefing do cliente e escopo
├── proposta/          — documentação de proposta e acordos
├── entregaveis/       — assets finais, screenshots, deploy
└── CLAUDE.md          — contexto deste projeto
```

## Como trabalhar aqui

**Setup inicial:**
```bash
cd projeto
npm install  # ou: bun install
npm run dev  # para rodar localmente
```

**Estrutura de código:**
- `src/` — componentes React, páginas, utilitários
- `public/` — assets estáticos (imagens, ícones)
- `tailwind.config.ts` — configuração de design tokens
