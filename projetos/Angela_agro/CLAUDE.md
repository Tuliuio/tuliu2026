# Agro Aplicação — Link de Bio

## O que é

Angela é especialista em tecnologia de aplicação de defensivo agrícola. Trabalha com produtores rurais para garantir aplicações corretas, sem desperdício, com máxima eficiência e mínima contaminação. O projeto começa com um **link de bio** — porta de entrada que apresenta o serviço e leva a uma conversão para domínio personalizado, site completo, landing page, automações e agentes futuros.
