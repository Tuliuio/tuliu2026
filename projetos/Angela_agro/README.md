# Link de Bio — Agro Aplicação

## Arquivos

- **`index.html`** — Versão HTML pronta pra publicar (inclui Tailwind via CDN)
- **`link-bio.jsx`** — Componente React/JSX para Lovable ou Next.js
- **`assets/`** — Logo e outros ativos visuais

## Como usar

### Opção 1: HTML puro
1. Abra `index.html` no navegador
2. Teste no mobile (Chrome DevTools → toggle device toolbar)
3. Atualize o número do WhatsApp em `href="https://wa.me/..."`
4. Atualize o email e configurações de integração conforme necessário
5. Publique na Hostinger com `/hostinger-deploy`

### Opção 2: Lovable
1. Copie o código do `link-bio.jsx`
2. Cole em novo projeto no Lovable
3. Configure as integrações (WhatsApp, email, calendário)
4. Deploy direto pelo Lovable
